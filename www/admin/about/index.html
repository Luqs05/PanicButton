About

The Safety Circle system was built to aid in the use of a panic system initiated in Alaska, Mamelodi. In Alaska, a Non-Profit Organization named Viva Foundation assists in the implementation and maintenance of a response initiative called SAPRI which aims at responding to sexual and domestic violence.

The initiative is said to have decreased the total amount of domestic and sexual violence in the area tremendously. To aid in further asccesibility of the SAPRI initiative, Safety Circle was created to provide a panic button and response system to users in Alaska, and possibly further. For these reasons, three system components have been developed:

  • Mobile Web App: Panic Button
  • Mobile Web App: Responder Map
  • Chrome App/Web App: Desktop Administrator Dashboard

Panic Button

Merely open the link in an emergency, ensure that your location settings are turned on and that there is a stable internet connection; Press the help button and if you have enough time to report the type of incident then proceed to select whether it is a criminal activity, medical emergency and/or environmental incident. If you are not provided with enough time to select the type of incident, do not fear- your location has been logged once you pushed the help button

Responder Map

To be able to access the responder map, you must contact the system administrator to grant you access. After registration, every time an incident is reported, you will receive an email stipulating that the button has been pushed with a link to a map, from here you may view all unattended cases near you. A long press on the map provides a context menu that allows the user to place a start and end point on the map for routing.

Desktop Administrator Dashboard

The Administrator Page allows Viva Foundation to have a live feed of the reported cases on a dashboard that makes use of a websocket for instantaneous results. The edit tab allows the administrator to make changes to the data by adding, updating or deleting points. On both of these pages, the admin is greeted by a table on which you can click to zoom to a point or click a point to view the table entry.
